Ch8: Forms

  • Accessing Form Elements:
    Instead of using a numerical index, we can use the name attribute to identify a form.
  • A form object also has a method called elements that returns an HTML collection of all the elements contained in the form. 
    const [input,button] = form.elements;

// or the name attributes:
const input = form.searchInput
const input = form['searchInput']
  • Properties and methods: 
    <button type='submit'>Submit</button> 
<input type='submit' value='Submit'>
<input type='image' src='button.png'>
form.action = '/an/other.url'
  • Form events:
    • focus: 
      input.addEventListener('focus', () => alert('focused'), false);
    • blur: 
      input.addEventListener('blur', () => alert('blurred'), false);
    • change: 
      input.addEventListener('change', () => alert('changed'), false);
  • Submitting a form: 
    // Add event lister to intercept the form
const form = document.forms['search'];
form.addEventListener ('submit', search, false);

function search() {
  alert(' Form Submitted');
}

// Stop the form from being submitted to the sample url
function search(event) {
  alert('Form Submitted');
  event.preventDefault();
}
  • Retrieving and changing values from a form: 
    // Retrieving example:
function search(event) {
    alert(`You Searched for: ${input.value}`);
    event.preventDefault();
}

// Setting example:
input.value = 'Search Here';

// Using event listener to improve UX:
input.addEventListener('focus', function(){
  if (input.value==='Search Here') {
      input.value = '' 
  }
}, false);

input.addEventListener('blur', function(){
  if(input.value === '') {
      input.value = 'Search Here';
  } }, false);

//OR you can use the placeholder attribute for the UX:
<input type='text' name='search-box' placeholder='Search Here'>
  • Form Controls:
    • input (contains attributes: autofocus, placeholder, maxlength)
    • select
    • textarea
    • button
  • Input fields:
    • text -- (type='text')
    • password -- (type='password')
    • checkbox -- (type='checkbox')
    • radio button -- (type='radio')
    • hidden input -- (type='hidden') [it is not secret, just hidden, so it is not safe for storage of sensitive data]
    • file input -- (type='file')
    • other types -- (number, tel, color, email, textarea)
    • select dropdown -- (select and then option)
    • buttons -- (type='reset', type='button', type='menu', type='menuitem', type='li')
