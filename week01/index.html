Week 1 Notes

Master Mobile UX

How to Rewire Your Brain for Mobile First UX

  • We need to think mobile first to improve user experience (adapting to new technologies).
  • New devices are being introduced at a fast pace.
  • Mobile first is not the same as responsive design.
  • Start small.

How to Improve UX by Understanding Users

  • Simplify actions to make the users happy.
  • Understand the user behaviors, such as how they handle their devices.
  • Keep performance on mind (how fast pages load)
  • You can reduce image usage. You can also optimize and minify CSS and JS files. GZIP files to elimante unnecesary data.

Better Mobile UX with Strategic Meny Design

  • Rechability: We could move navigation bars to better location (Facebook app is an example).
  • Perception: Some users don't know what the icons mean.

Designing Intuitive and Mobile Friendly Forms

  • Focus on the important things first! Make users think less.
  • Float label are an option to simplify form inputs.
  • Pagination is another option for larger forms.

Designing a Better Mobile Advertising User Experience

  • Research before advertising.
  • Avoid pop-ups/pop-overs for mobile.
  • Make ads small and simple. Content first!
